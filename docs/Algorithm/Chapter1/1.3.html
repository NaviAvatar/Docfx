    3.求整数n的阶乘的算法如下，计算时间复杂度：

    int fact(int n)
{
    if(n <= 1){
      return 1;  
    } 
    return n*fact(n-1);
}

    这是个递归方法计算n的阶乘。显然一共调用fact函数n次，故其算法时间复杂度为O(n);
